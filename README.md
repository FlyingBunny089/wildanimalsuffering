# wildanimalsuffering

## Running Locally
1. `npm run watch`
2. `cd public`
3. `python3 -m http.server 8000`

## Tech Stack
Vanilla JS + [Webpack Mix](https://github.com/devanandb/webpack-mix) + Python for serving content
